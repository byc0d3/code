MariaDB

MariaDB es un sistema de gestión de bases de datos derivado de MySQL con licencia GPL (General Public License).

RHEL

Estas instrucciones estan garantizadas para Sistemas Operativos basados en RHEL como Rocky Linux, Almalinux y Centos.

Instalación 🔧

1 - Descargamos la ultima versión de mariadb:

curl -LsS https://downloads.mariadb.com/MariaDB/mariadb_repo_setup | bash

1 - Instalamos MariaDB:

sudo dnf install mariadb-server

Versión 📦

2 - Para ver la version de MariaDB:

sudo mariadb -V

Comandos basicos ⚙️

3 - Activar inicio automatico del Servicio:

sudo systemctl enable mariadb

4 - Iniciar Servicio:

sudo systemctl start mariadb

5- Status de Servicio:

sudo systemctl status mariadb

6 - Reiniciar Servicio:

sudo systemctl restart mariadb

7 - Detener Servicio:

sudo systemctl stop mariadb

8 - Desactivar inicio automatico del Servicio:

sudo systemctl disable mariadb

Querys Basicos 🔧

1 - Nos conectamos la base de datos:

sudo mariadb -u root -p

2 - Crear base de datos con utf8_general:

CREATE DATABASE database CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

3 - Creamos un usuario:

CREATE USER 'user'@'%' IDENTIFIED BY 'passwd';

4 - Le damos permisos a ese usuario sobre esa base de datos:

GRANT ALL PRIVILEGES ON database.* TO 'user'@'%';

5 - Para el usuario Administrador que tendra permiso some todas las bases de datos:

GRANT ALL PRIVILEGES ON *.* TO 'user'@'%' WITH GRANT OPTION;

6 - Permiso a usuario solo de CRUD sobre esa base de datos:

GRANT SELECT, INSERT, UPDATE, DELETE, CREATE, DROP ON database.* TO 'user'@'%';

5 - Para el usuario Administrador que tendra permiso some todas las bases de datos:

GRANT ALL PRIVILEGES ON *.* TO 'user'@'%' WITH GRANT OPTION;

7 - Recarga los privilegios:

FLUSH PRIVILEGES;
