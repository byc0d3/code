Nmcli

nmcli (NetworkManager Command Line Interface) es una potente utilidad de línea de comandos para gestionar conexiones de red en Linux.

RHEL

La configuración específica de las redes en un servidor dependerá del uso que le vayas a dar, estos son pasos basicos que por lo general se usan en todos los servidores.

Información de las interfaces de red 🔧

1️⃣ - Primero listaremos todas las interfaces de red en nuestro sistema:

sudo nmcli device show

2️⃣ - Si queremos ver información mas detallada de las interfaces:

sudo nmcli -p device show

3️⃣ - Para conocer cuales interfaces de red estan activas:

sudo nmcli con show

Crear redes 🔧

4️⃣ - Crear una nueva conexión de red con DHCP:

sudo nmcli con add con-name ensxxx type ethernet ifname ensxxx ipv4.method auto

5️⃣ - Crear una nueva conexión de red con dirección estatica:

sudo nmcli con add con-name ensxxx type ethernet ifname ensxxx ipv4.method manual ipv4.address 192.168.1.102/24 ipv4.gateway 192.168.1.1

6️⃣ - Para activar esta interfaz:

sudo nmcli con up ensxxx

7️⃣ - Añadir un servidor DNS a una conexión existente:

sudo nmcli con mod "ensxxx" ipv4.dns 8.8.8.8,8.8.4.4

8️⃣ - Desactivar la dirección IPv6 con nmcli:

sudo nmcli con mod "ens35" ipv6.method ignore

9️⃣ - Modificar una conexion existente:

sudo nmcli connection modify ensxxx ipv4.addresses 192.168.1.103/24

9️⃣.1️⃣ - Activar esta interfaz:

sudo nmcli con up ensxxx

9️⃣ - Agregar una ip a una interfaz, con IPv6 desactivado sin gateway:

sudo nmcli con add con-name ensXXX type ethernet ifname ensXXX ipv4.method manual ipv4.address 10.31.196.XXX/25 ipv6.method disabled

🔟 - para eliminar una conexion existente:

sudo nmcli con delete ensxxx

SELinux (Security-Enhanced Linux) es un módulo de seguridad para el kernel de Linux que proporciona una capa adicional de seguridad sobre el sistema tradicional de control de acceso discrecional (DAC).

Archivos de Configuración 🔧

1️⃣ - En la etapa de configuración inicial del servidor, es recomendable establecer el SELinux en modo Permissive (para aplicar los cambios, deberá reiniciar su servidor), editamos:

sudo nano /etc/sysconfig/selinux

2️⃣ - Modificamos la siguente linea:

SELINUX=permissive

3️⃣ - Reiniciamos el sistema:

sudo reboot

La zona horaria es una región de la Tierra que tiene la misma hora estándar para fines legales, comerciales y sociales. Es una forma de estandarizar la medición del tiempo en diferentes ubicaciones geográficas, a medida que la Tierra gira y la posición del sol cambia.

Configurar Zona Horaria 🔧

1️⃣ - Para cambiar la hora y la zona horaria ejecutamos el siguiente comando, modificando segun sea el Pais/Estado:

sudo timedatectl set-timezone America/Caracas

2️⃣ - Si desea ennumerar todas las zonas horarias disponibles ejecutamos lo siguiente:

sudo timedatectl list-timezones

3️⃣ - Para sincronizar el tiempo, y habilitarlo ejecutamos::

sudo systemctl start chronyd && systemctl enable chronyd && systemctl status chronyd

Un banner al iniciar sesión por SSH en tu servidor es una excelente manera de mostrar información importante, como avisos legales, políticas de uso o simplemente un mensaje de bienvenida personalizado.

Configurar Banner 🔧

1️⃣ - Para crear un banner que sea visible al conectarnos por ssh, editamos el siguiente archivo:

sudo nano /etc/motd

2️⃣ - Agregamos cualquier dieño, es este caso dejo esta plantilla:


                        |----------------------------------------------------------------|
                        | """                     SERVER-APP01-PRO                    "" |
                        | -------------------------------------------------------------- |
                        | -------------------------------------------------------------- |
                        | """ Si no tiene autorizacion para ingresar a este servidor """ |
                        | """ salga de inmediato, el mismo esta siento monitoriado   """ |
                        | """ se aplicaran severas sanciones por uso no autorizado.  """ |
                        | -------------------------------------------------------------- |
                        | -------------------------------------------------------------- |
                        | """    SEDE -  Direccion - Infraestructura - Servidores    """ |
                        | -------------------------------------------------------------- |

3️⃣ - Guardamos y salimos, al conectarnos nuevamente a servidor se vera el banner en la consola
